# 项目简介 python647

本项目是一个基于Python Django框架的图书管理系统。

## 技术栈

- 后端：Python Django
- 数据库：MySQL

## 功能模块

- 用户认证
  - 登录
  - 注册
  - 登出
  - 修改密码
- 数据管理
  - 添加图书
  - 添加作者
  - 添加出版社
  - 查找出版社信息
  - 查看图书信息
- 安全性
  - 密码加密
  - 信息唯一性判断

## 系统角色

- 系统管理员
- 普通用户

## 运行环境

- Python 3.6+
- Django 2.2+
- MySQL 5.7+

## 部署步骤

具体部署步骤请参考项目附带的部署文档。

## 目录结构

```plaintext
 project/
 ├── apps/                 # 业务逻辑代码目录
 ├── db/                  # 数据库相关文件目录
 ├── static/              # 静态文件目录
 ├── templates/           # 模板文件目录
 ├── manage.py             # 管理脚本
 └── README.md             # 项目说明文档
```

## 核心亮点

- 使用Django框架，开发效率高，易于维护。
- 支持多种用户操作，满足日常图书管理需求。
- 加密用户密码，确保数据安全。
- 实现信息唯一性判断，保证数据的准确性。
